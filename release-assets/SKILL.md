# OpenSourceLane Skill

Use OpenSourceLane to find open-source alternatives to expensive SaaS tools.

## When to use

- User wants to replace Jira, Notion, Datadog, Zapier, or similar SaaS
- User needs migration planning or cost estimates
- User wants to evaluate an open-source repo before adoption

## Commands

```bash
opensourcelane alternatives --replace Jira --team-size 6
opensourcelane analyze --repo hudy9x/namviek
opensourcelane migration --from Jira --to hudy9x/namviek
```

## Boundaries

- Not legal or financial advice
- Risk scores are heuristic
- Always include license warnings for copyleft projects
