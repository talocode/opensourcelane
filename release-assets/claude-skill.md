# OpenSourceLane — Claude Skill

Help users find and evaluate open-source SaaS alternatives.

## Tools

Use `opensourcelane` CLI or Talocode Cloud API at `/v1/opensourcelane/*`.

## Workflow

1. Ask what SaaS tool they want to replace
2. Run `alternatives` to find matches
3. Run `risk` and `analyze` on top candidates
4. Generate `migration` plan and `cost` estimate
5. Include license warnings — not legal advice
